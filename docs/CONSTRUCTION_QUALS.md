# 建筑工程资质速查表

本仓 `/tyc-construction-bid` SKILL 对建筑企业进行四维核验，数据直连国家"四库一平台"。

## 四维核验

```
┌────────────────────────────────────────────────────────┐
│            建筑工程投标资格四维                          │
├────────────────────────────────────────────────────────┤
│                                                         │
│   ① 资质证书       ② 注册人员                          │
│   (get_construction_qualifications)                     │
│      ┃                (get_construction_registered_personnel)│
│      ▼                     ▼                            │
│   ┌──────────────────────────────┐                     │
│   │     项目可投标资格            │                     │
│   └──────────────────────────────┘                     │
│      ▲                     ▲                            │
│      ┃                     ┃                            │
│   ③ 工程业绩        ④ 不良行为                          │
│   (get_construction_projects)                           │
│                   (get_construction_bad_conduct)        │
│                                                         │
└────────────────────────────────────────────────────────┘
```

## 详细字段

### ① 资质证书（get_construction_qualifications）

| 字段 | 说明 |
|------|------|
| 资质类别 | 施工总承包 / 专业承包 / 施工劳务 / 设计 / 监理 |
| 资质等级 | 特级 / 一级 / 二级 / 三级 |
| 资质证书号 | 官方编号 |
| 发证机关 | 住建部 / 省厅 / 市局 |
| 有效期 | 起 / 止时间 |

### ② 注册人员（get_construction_registered_personnel）

| 字段 | 说明 |
|------|------|
| 姓名 | 注册建造师 / 工程师 / 监理 |
| 专业类别 | 建筑 / 市政 / 公路 / 水利 / 电力 / 机电 等 |
| 等级 | 一级 / 二级 |
| 执业印章号 | 官方印章编号 |
| 注册单位 | 是否挂靠本企业 |

**挂靠风险判断**：若同一人员同时注册在多家企业（兼任），可能存在挂靠或借证嫌疑。

### ③ 工程业绩（get_construction_projects）

| 字段 | 说明 |
|------|------|
| 项目编号 | 国家平台唯一编号 |
| 项目名称 | 工程名 |
| 属地 | 省 / 市 / 区 |
| 项目类别 | 房建 / 市政 / 公路 / 水利 等 |
| 中标日期 | 招投标日 |
| 合同金额 | 元 |
| 建设单位 | 甲方 |

### ④ 不良行为（get_construction_bad_conduct）

| 字段 | 说明 |
|------|------|
| 行为类型 | 违规施工 / 安全事故 / 质量问题 / 投标违规 |
| 处罚机关 | 建委 / 住建局 |
| 处罚日期 | 年月日 |
| 处罚措施 | 警告 / 罚款 / 停标 / 吊销资质 |
| 影响期 | 至年月日 |

## 投标资格建议

| 情况 | 建议 |
|------|------|
| 资质齐全 + 人员齐备 + 业绩达标 + 无不良 | ✅ 通过 |
| 资质齐全 + 人员齐备 + 业绩达标 + 不良（轻）| 🟡 附加条款 |
| 资质缺项 或 人员不足 | ⚠️ 补件后再议 |
| 严重不良（停标 / 吊销资质）| ❌ 拒绝 |

## 常见资质分级（供参考）

| 类别 | 等级 | 最高可承接金额 |
|------|------|-------------|
| 施工总包 特级 | 特级 | 无限制 |
| 施工总包 一级 | 一级 | 单项合同 ≤ 企业注册资本 × 5 |
| 施工总包 二级 | 二级 | 单项合同 ≤ 1 亿元（房建）|
| 施工总包 三级 | 三级 | 单项合同 ≤ 2000 万（房建）|

注：具体金额限制以《建筑业企业资质管理规定》最新版为准。

## 数据覆盖

TYC 建筑资质数据直连国家"四库一平台"（住建部主办），覆盖全国所有持证建筑企业。
