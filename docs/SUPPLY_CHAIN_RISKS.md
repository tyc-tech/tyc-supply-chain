# 18 类供应链风险清单

## 严重度判级规则

- 🔴 高：立即风险（建议暂停合作 / 催收 / 调档）
- 🟡 中：关注风险（加强监控 / 合同条款强化 / 补保证金）
- 🔵 信息：中性信号（仅记录，不影响准入）

## 综合风险评级

| 高风险数 | 中风险数 | 综合评级 |
|---------|---------|---------|
| ≥3 | 任意 | 🔴 严重 |
| 1-2 | ≥2 | 🔴 严重 |
| 1 | 0-1 | 🟡 中度 |
| 0 | ≥3 | 🟡 中度 |
| 0 | 1-2 | 🔵 轻度 |
| 0 | 0 | 🟢 良好 |

## 建议动作

| 综合评级 | 建议 |
|---------|------|
| 🔴 严重 | 暂停合作 / 加速催收 / 转替代供应商 |
| 🟡 中度 | 加强跟踪（月度） / 补强保证金 / 缩短账期 |
| 🔵 轻度 | 季度跟踪 / 维持现有合作 |
| 🟢 良好 | 常规年度复评 |
